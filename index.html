About The Workshop

This intensive workshop is designed for engineering students at MITS to bridge the gap between academic concepts and real-world technological applications. Participants will gain hands-on experience in AI, data visualization, and generative models by building practical projects from scratch. This workshop is tailored for students in CSE, AI, Data Science, and ECE to equip them with skills demanded by the modern tech industry.

Concept: The Rise of AI Agents and Task Automation. We'll explore AI agents like Siri, their core components (language interpreter, execution engine), and relevance for RPA.

Project: Create a Python-based personal desktop assistant that takes voice/text commands to open apps, tell the time, or search Wikipedia.

Skills Gained: AI-driven automation, command processing, Python scripting with libraries like SpeechRecognition and pyttsx3.

Concept: The Art and Science of Data Storytelling. Learn principles of effective visualization and an introduction to Pandas and Plotly libraries.

Project: Build an "MITS Insights" interactive dashboard using an anonymized, prepared dataset to visualize trends in placements, diversity, and club participation.

Skills Gained: Data manipulation with Pandas, creating interactive charts (bar, pie, timeline) with Plotly, data analysis.

Concept: Introduction to Generative AI and Large Language Models (LLMs). We'll cover how LLMs work, prompt engineering, and leveraging APIs for app development.

Project: Build a "Creative Story Generator" web app using the Hugging Face `transformers` library and a pre-trained model like DistilGPT-2, with a simple UI built using Streamlit.

Skills Gained: Generative AI integration, working with pre-trained models, API usage, and rapid UI development with Streamlit.
